# Project Overview

This is a NestJS application named `nestjs-story-app`. It appears to be a backend service that handles stories, payments (MercadoPago), and potentially file storage (AWS S3) and notifications (Firebase Admin, Nodemailer). It uses TypeScript.

# Building and Running

## Installation

To install the project dependencies, run:

```bash
npm install
# or
yarn install
```

## Building

To build the project, use:

```bash
npm run build
# or
yarn build
```

## Running

### Development Mode

To run the application in development mode with hot-reloading:

```bash
npm run start:dev
# or
yarn start:dev
```

### Production Mode

To run the compiled application in production mode:

```bash
npm run start:prod
# or
yarn start:prod
```

# Testing

To run the tests:

```bash
npm run test
# or
yarn test
```

# Development Conventions

## Linting

To lint the codebase and fix issues:
